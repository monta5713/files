Voor het runnen van de applicatie: 

Om het applicatie te gebruiken moet u eerst de statements.sql bestand importen via de localhost. 
Nadat u de database heeft aangemaakt worden er een paar voorbeelden toegevoegd. Dit zijn aanpasbaar. 
Daarna moet u de populate.php file een keer runnen via uw browser zodat er een admin user wordt aangemaakt 
met de username: medewerker en password: medewerker. Er wordt ook een klant account gemaakt met een username: test en password: test. 
Hierna kunt u inloggen via de login.php voor de medewerker en login-klant.php file voor de klant.

Omdat er niet naar gevraagd wordt om meerdere medewerkers toevoegen, verwijderen of wachtwoord wijzigen 
van de admin account, heb ik daar geen functionaliteiten toegevoegd.

Vervolgens moet een klant een account aanmaken via de register pagina om een reservering te doen.
